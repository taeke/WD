The IGame type exposes the following members.

Methods

  NameDescription
AddPlayer
Add's a new IPlayer to the IGame.
Attack
The currentPlayer wants to attack a Country of another [!:Iplayer].
ChooseTurnType
Each turn the currentPlayer must decided which kind of TurnType he wants to play.
ExchangeCards
Each turn the currentPlayer can decided to exchanges a CardType collection for armies if he has a valid combination.
GoToNextFase
The currentPlayer wants to go to the next GameFase.
MoveArmies
The currentPlayer can move armies from one Country to another Country he owns.
PlaceNewArmies
The currentPlayer wants to place new armies on a Country he owns.
StartGame
After all the IPlayer are added the game can start with this method.
